## Description
Sync Metronome lets your entire band hear the same perfectly synchronized click track during practice. One person hosts, everyone else joins — all hearing the exact same beat at the exact same time.

No more counting in. No more drifting apart. Just press play and stay locked in.

FEATURES:
- Real-time sync across all connected devices
- Adjustable BPM (40-240) with +1/+5 controls
- Time signature support (4/4, 3/4, 2/4, 6/8)
- Works over WiFi or mobile hotspot (tethering)
- Solo mode available offline
- Screen stays on during playback
- Clean, distraction-free interface

HOW IT WORKS:
1. One band member opens the app and taps "Host"
2. Other members tap "Join" and select the room
3. The host sets the BPM and presses play
4. Everyone hears the click in perfect sync

Connect via the same WiFi network, or have one member share a mobile hotspot. Works with any mobile data connection (4G/5G).

Built for musicians, by musicians.
